## Structure

```
index.html          Home page linking to all projects
css/style.css       Single shared stylesheet for the whole site
project1/           Project 1 pages and optimized gallery images
project2/ – 4/      Placeholders for upcoming projects
reflection-draft.md Project reflection notes
```

## Notes

- All pages use semantic HTML5 with a skip-to-content link and ARIA-labeled navigation for accessibility.
- Styling comes from one external stylesheet shared across every page.
- Project 1 focuses on search engine optimization best practices, accessible markup, and an image gallery with optimized thumbnails.

## Viewing

Open `index.html` in a browser — no build step or server required.
